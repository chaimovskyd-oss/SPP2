# מערכת Bounds ו־Hit Testing

מספקת axis-aligned bounds, rotated bounds, transformed bounds, group/page bounds, selection handles ו־hit testing לפי z-index.

משתמשים בה: selection, snapping, marquee, transform handles ו־export validation.

אסור לעשות: לחשב bounds ידנית בכל קומפוננטה, להתעלם מרוטציה, או לבחור שכבות נעולות/נסתרות בלי policy מפורשת.

הרחבה עתידית: polygon hit testing, masks, nested group transforms ו־click-through מתקדם.
