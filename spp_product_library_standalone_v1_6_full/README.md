# SPP Product Library Standalone v1.6 Full

Run:
```bash
pip install -r requirements.txt
python main.py
```

Includes:
- Full product data from your Base44 export
- Product image URLs for thumbnails + local thumbnail cache
- Search / filters / thick scrollbars
- Edit Product: dimensions, bleed, printer type, heat press instructions
- Batch Edit: selected products or full category
- Masks: attach, replace, remove, preview
- Export for SPP includes bleed, safe area, production instructions and mask path

Note:
Thumbnail images are downloaded from the saved product URLs the first time the app runs on a computer with internet access.
If internet is unavailable, gray placeholders are shown instead of crashing.
